---
name: ui-design
description: Create modern, clean, responsive UI using Tailwind and good UX principles.
---

Always create polished modern UI.
Use spacing generously.
Prefer premium SaaS aesthetics.
Use TailwindCSS and responsive layouts.
Use shadcn/ui when possible.
Avoid ugly default HTML styling.

When generating UI:

- Use modern SaaS design patterns
- Prefer clean spacing and visual hierarchy
- Use rounded corners and soft shadows
- Mobile-first responsive layouts
- Use TailwindCSS
- Keep components reusable
- Prioritize accessibility
- Avoid cluttered interfaces
- Use consistent typography scale
- Prefer subtle animations
- Follow modern dashboard aesthetics
- Use cards/grids instead of long forms
